漢字テストメーカー

小学校1年生から中学校3年生までの漢字テストを、ブラウザ上で簡単に作成・印刷できるWebアプリケーションです。
先生方の定期テスト作成業務の効率化を支援します。

主な機能

学年別出題: 小1〜中3までの学年別、または「小学全」「中学全」から範囲を選択可能。

出題形式: 「読み（漢字→かな）」「書き（かな→漢字）」の切り替えに対応。

レイアウト: 国語のテストで一般的な「縦書き」と、使いやすい「横書き」に対応。

問題数設定: 10問、20問、50問から選択可能（20問以上は自動でページ分割）。

自由入力モード: 独自の漢字・読み・例文を入力して、オリジナルのテストを作成可能。

編集機能: 自動生成された問題文や答えを、プレビュー画面で個別に修正可能。

印刷・PDF保存: A4サイズに最適化されたレイアウトで、ブラウザから直接印刷やPDF保存が可能。

模範解答: テスト用紙とは別に、答え合わせ用の模範解答ページも自動生成。

使い方

設定:

画面上の設定パネルで「学年」「出題形式」「レイアウト」「問題数」を選択します。

独自の問題を作りたい場合は「自由入力」を選択し、漢字,よみ,例文 の形式でデータを入力します。

作成:

「テストを作成する」ボタンをクリックすると、ランダムに問題が抽出・生成されます。

確認・編集:

プレビュー画面が表示されます。

「問題入替」ボタンで、問題を再シャッフルできます。

「内容編集」ボタンを押すと、問題文や解答を自由に書き換えられます（例文中の答え部分は [ ] で囲んでください）。

印刷:

「印刷/保存」ボタンをクリックします。

ブラウザの印刷ダイアログが開きます。「送信先」をプリンターにすれば紙に印刷、PDF形式を選べばファイルとして保存できます。

動作環境

最新のWebブラウザ（Google Chrome, Edge, Safari, Firefoxなど）

インターネット接続（ReactなどのライブラリをCDNから読み込むため）

インストール方法

本アプリは単一のHTMLファイルで動作します。
index.html をダウンロードし、お使いのブラウザで開くだけで使用可能です。特別なサーバーやビルド環境は不要です。

技術スタック

HTML5

React 18 (CDN)

Tailwind CSS (CDN)

Babel (Standalone)

注意事項

印刷設定: 印刷時は「背景のグラフィック」を有効にする必要はありませんが、レイアウト崩れを防ぐため「用紙サイズ：A4」「倍率：100%（または用紙に合わせる）」の設定を推奨します。

データ保存: 作成したテストデータはブラウザ内に一時的に保持されますが、ページを再読み込みすると初期化されます。長期保存したい場合はPDFとして保存してください。
